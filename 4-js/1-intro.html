Présentation Javascript

Le javascript est un langage de programmation : il permet de coder des algorithmes
Il peut être exécuter dans un navigateur : il va permettre de dynamiser cette page. Le contenu va pouvoir évoluer après le chargement de la page.

Il est exécuté côté client : le code peut être exécuter sans serveur (échange client / serveur).

Pour écrire du javascript :

  • On l'écrit directement dans une balise script
  • On l'écrit dans un fichier .js à part, et on inclut ce fichier dans la page

On place en général le javascript en bas du code, parce qu'il est bloquant. On affiche le HTML/CSS avant de commencer à exécuter du code JS. On place le JS juste avant la fermeture de la balise body.
