CSS : Positionnement Flexbox

  • display:flex pour le container
  • flex: flex-grow flex-shrink flex-basis
  • flex-grow : facteur d'agrandissement
  • flex-shrink : facteur de rétrécissement
  • flex-basis : largeur des éléments
  • flex-wrap : sur le container / retour à la ligne ou non des éléments qui dépassent
  • justify-content: aligner les éléments avec ou sans espaces, etc.
  • justify-direction: en ligne ou en colonne / dans l'ordre ou dans l'ordre inverse
